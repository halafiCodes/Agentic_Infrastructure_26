# Tenx MCP Sense Setup (Local)

If the extension is not listed in the public marketplace, it is likely private. Use one of these methods:

## 1) Private Marketplace
- Sign in to the organization account in VS Code.
- Open Extensions (Ctrl+Shift+X) and search for the internal name provided by your org.
- Install and reload.

## 2) VSIX Install
- Obtain the .vsix file from your org.
- In VS Code: Extensions → "..." menu → Install from VSIX...
- Reload VS Code.

## Verify Connection
- Open the Output panel.
- Select "Tenx MCP Sense" from the dropdown.
- Confirm logs show **Connected** or **Session started**.
